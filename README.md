## Requirements

* Requires iOS 8.0 or later

## Installation

BDAPI is available through [CocoaPods](https://cocoapods.org/pods/BDAPI). To install it, simply add the following line to your Podfile:

```
pod 'BDAPI'
```

## Usage

```
#import <BDAPI/BDAPI.h>
```
